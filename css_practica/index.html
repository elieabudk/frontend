El modelo de caja CSS es básicamente un cuadro que envuelve cada elemento HTML. Está formado por: conte nido, relleno, bordes y márgenes. La siguiente imagen ilustra el modelo de caja:

Contenido : el contenido del cuadro, donde aparecen el texto y las imágenes. Relleno : limpia un área alrededor del contenido. El relleno es transparente. Borde : un borde que rodea el relleno y el contenido. Margen : limpia un área fuera del borde. El margen es transparente.
